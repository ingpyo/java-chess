# 기능 목록
- [X] 체스 보드
  - [X] 8 x 8칸의 정사각형 이다.
    - [X] 각 칸은 좌표가 있다. x좌표(a-h), y좌표(1-8)
  - [X] 기물을 초기위치에 배치시킨다.

- [X] 기물
  - [X] 진영이 존재한다.(흑, 백)
  - [X] 계급이 존재한다.

- [X] 체스 보드 출력
  - [X] 기물은 진영에 따라 다르게 출력한다.
    - [X] 흑은 대문자로 출력한다.
    - [X] 백은 소문자로 출력한다.
  - [X] 비어있는 칸은 "."으로 출력한다.
